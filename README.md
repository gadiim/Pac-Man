
# Pac-Man

## Objective
Create a Pac-Man game using JavaScript, HTML, and CSS.

## Tools
- JavaScript
- HTML
- CSS
- Node.js
- GitHub Actions (for CI/CD)
- GitHub Pages (for hosting)

## Project Structure
   ```sql
Pac-Man/
├── .github/

├── src/
│ ├── assets/

│ │ └── powerPellet.js
│ ├── collisions/

  ```
## Scripts
- `npm start`: Start the development server.
- `npm install`: Install dependencies.
